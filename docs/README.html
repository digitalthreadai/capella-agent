// Quick Start

Up and Running in Minutes

Clone, build, install, and start querying your Capella models with natural language.

quickstart.sh 
# Build
$ git clone <repo-url>
$ cd capella-agent
$ mvn clean verify

# Install in Capella
# Help > Install New Software > Add > Local
# > com.capellaagent.site/target/repository

# Configure LLM
# Preferences > Capella Agent > LLM Provider > Set API key

# Use
# Window > Show View > Capella Agent > AI Model Chat
# Type: "List all system functions in SA layer"
// Project Structure

9-Project Plugin Architecture

Modular OSGi bundles following Eclipse plugin best practices with clean separation of concerns.

capella-agent/ 
com.capellaagent.core/            Core agent framework
  src/.../llm/                     LLM abstraction layer
  src/.../tools/                   Tool registry & base classes
  src/.../message/                 Message bus & routing

com.capellaagent.chat/            Model Chat Agent (16 tools)
  src/.../tools/read/              Model query tools
  src/.../tools/write/             Model mutation tools
  src/.../tools/diagram/           Diagram management

com.capellaagent.teamcenter/      Teamcenter Agent (7 tools)
  src/.../rest/                    REST client & auth
  src/.../tools/                   PLM integration tools

com.capellaagent.simulation/      Simulation Agent (6 tools)
  src/.../matlab/                  MATLAB engine bridge
  src/.../tools/                   Simulation tools

com.capellaagent.mcp/             MCP Server Bridge (Claude Code)
  src/.../mcp/                     HTTP endpoint & tool adapter
  src/.../mcp/bridge/              Standalone stdio bridge

com.capellaagent.ui/              Eclipse UI views & dialogs
com.capellaagent.feature/         Eclipse feature project
com.capellaagent.site/            P2 update site
com.capellaagent.target/          Target platform definition
