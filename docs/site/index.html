Get Running in 3 Steps

1

Install the Plugin

Run reinstall.bat from the repository root (sets CAPELLA_HOME first), then restart Capella. Or use Help → Install New Software with the p2 site.

2

Configure a Provider

Open Window → Preferences → Capella Agent → LLM Providers. GitHub Models is free and requires only a GitHub token — no billing setup.

3

Open the IFE Model and Ask

Press Ctrl+Shift+M to open the AI Chat view. Type List all physical functions and press Enter.
