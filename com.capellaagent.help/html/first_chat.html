Your First Chat

This walkthrough uses the In-Flight Entertainment (IFE) sample model bundled in the repository under demo/. If you have your own Capella project open, the same steps apply.

Step 1 — Open the IFE Sample Model

  1. In the Project Explorer, right-click in an empty area and choose Import... → General → Existing Projects into Workspace.
  2. Browse to the demo/ folder in the repository and import the IFE project.
  3. Double-click the .aird file to open the Sirius session. You should see the 5 ARCADIA layers (OA, SA, LA, PA, EPBS) in the Project Explorer tree.

Step 2 — Open the AI Model Chat View

Press Ctrl+Shift+M or go to Window → Show View → Other... → Capella Agent → AI Model Chat.

The view docks to the right side of the workbench by default. You should see:

  • A toolbar at the top with buttons for mode, model/provider selection, and utilities.
  • A browser area (the main chat window) showing either the welcome wizard or an empty-state card.
  • A text input field at the bottom with a placeholder prompt.

Confirm that the project dropdown in the toolbar shows IFE. If it shows "No open Capella sessions", make sure the .aird file is open.

Step 3 — Send Your First Query

In the input field, type:

List all physical functions

Press Enter to send.

You will see a brief tool-call indicator in the chat:

>> Executing tool: list_elements (layer=pa, element_type=functions) ...

After a second or two, the agent returns a sortable HTML table. Each row shows a physical function with its name, UUID, and a description excerpt.

Step 4 — Understand the Result Table

The result table has these features:

  • Sortable columns — click any column header to sort ascending or descending.
  • UUID links — each UUID in the table is a clickable hyperlink. Clicking one navigates your Project Explorer to that element and selects it.
  • Copy buttons — hover over any cell to reveal a small copy icon for copying the value to the clipboard.

Step 5 — Navigate to an Element via UUID

  1. Find a row that interests you — for example, a function called ProcessData.
  2. Click its UUID (looks like a1b2c3d4-...).
  3. The Project Explorer scrolls to and selects that element automatically.
  4. The Capella Properties view at the bottom of the workbench shows the element's details.
Tip: You can also ask the agent directly: Get details for element a1b2c3d4-... and it will call get_element_details to return the full property set for that UUID.
