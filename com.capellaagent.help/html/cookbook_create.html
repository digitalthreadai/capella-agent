Recipe: Create Model Elements

This recipe shows how to use Capella Agent to create new model elements using natural language. All write operations are recorded on the Eclipse undo stack. Estimated time: 15 minutes.

Prerequisites: IFE sample model open, General Assistant mode active.

This recipe modifies the IFE model. Use a copy of the model, or be prepared to undo changes with Ctrl+Z after each step.

Step 1 — Create a Physical Component

Create a physical component called "Sensor Hub" under the Avionics System

The agent calls create_element with the name "Sensor Hub", type PhysicalComponent, and parent resolved from "Avionics System" by name lookup.

The response includes a confirmation card showing:

  • The new element's UUID.
  • Its full path in the model hierarchy.
  • A "View in Project Explorer" link that clicks to navigate to it.

Step 2 — Verify in Project Explorer

Click the UUID link in the response. The Project Explorer scrolls to and selects Sensor Hub under the Avionics System node. Press F5 to refresh if the element does not appear immediately.

Step 3 — Add a Port

Add a port called "DataOut" to Sensor Hub

The agent calls create_port with name "DataOut" and the parent set to the Sensor Hub UUID from the previous step (held in conversation context).

Step 4 — Create a Functional Exchange

First, get the UUIDs of two existing physical functions you want to connect: 

List all physical functions containing "Process" or "Transmit" in their names

Then create the exchange:

Create a functional exchange between ProcessData and TransmitTelemetry

The agent calls create_exchange with the source and target resolved by name lookup. The exchange appears in the model's PA layer.

Step 5 — Undo All Changes

To revert the three changes (component, port, exchange), press Ctrl+Z three times, or type /undo three times in the chat input.

Multi-tool undo note: If the agent handled all three creations in a single turn (one LLM response that called three tools), each tool call is a separate Eclipse RecordingCommand. You still need to undo three times. This is a known limitation — see Known Limitations #1. A compound-undo-per-turn fix is planned for v1.1.

More Write Operations

What to sayTool calledNotes

"Create a capability called 'Sensor Acquisition'" create_capability Creates a CapabilityRealization in the current layer
