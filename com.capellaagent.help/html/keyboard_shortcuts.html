Keyboard Shortcuts

Capella Agent Shortcuts

ShortcutActionContext
Ctrl+Shift+M Open AI Model Chat view Global (Capella workbench)
Enter Send message Chat input field has focus
Shift+Enter Insert a new line (multi-line input) Chat input field has focus
Esc Cancel the running LLM request / tool execution Chat view (while a response is streaming)
Ctrl+L Clear the conversation (same as /clear) Chat view has focus
Ctrl+Z Undo the last model change (Eclipse standard undo) Global

Notes on Conflicts

Ctrl+E conflict: The Export toolbar button does not have a dedicated keyboard shortcut because Ctrl+E is already used by Capella for "Open Resource" (Quick Access). Use the export toolbar button or type /export in the chat input instead.
Re-mapping shortcuts: Capella Agent shortcuts can be customized in Window → Preferences → General → Keys. Search for "Capella Agent" to see all bindable commands.

Standard Eclipse Shortcuts Useful with Capella Agent

ShortcutActionWhy It Is Useful
Ctrl+Z Undo Reverses the last model change made by a tool call. For multi-tool turns, press multiple times.
Ctrl+Y Redo Reapplies a change after undo.
F5 Refresh Refreshes the Project Explorer after model changes.
Ctrl+Shift+R Open Resource Quickly navigate to any file in the workspace.
Ctrl+F6 Switch editor Cycle through open diagrams and editors.
