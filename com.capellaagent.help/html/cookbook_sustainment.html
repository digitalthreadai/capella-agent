Recipe: Sustainment Engineering Demo

This recipe walks through the full 5-minute IFE demo — the flagship showcase of Capella Agent. A sustainment engineer diagnoses a field fault using the live Capella model in under 60 seconds.

Target audience: Aerospace and defense sustainment engineering teams, Capella Days attendees, enterprise buyers evaluating the tool.

The Story Arc

It's 3am. A sustainment engineer receives a fault report from the field. Today, they open four different tools and spend 90 minutes identifying the right subsystem to dispatch. Watch the agent do it in 60 seconds.

Setup

  1. Open the IFE project in Capella with the .aird session running.
  2. Open the AI Model Chat view (Ctrl+Shift+M).
  3. Set the mode dropdown to Sustainment Engineer.
  4. Confirm the chat is empty (use /clear if needed).
  5. Verify the status bar shows "Ready" next to your provider name.

The Demo Query

Click the first starter prompt from the empty-state card, or type it manually:

The Seat TV in row 14 is rebooting every time we try to play a VOD movie. Where should I look?

Expected Agent Response Flow

The agent calls four tools in sequence:

Tool CallResult
classify_fault_ata ATA chapter 44 — Cabin Systems (shown as a badge in chat)
lookup_fault_symptom Candidate component types: Cabin Video Display Unit, Cabin Video Player, Cabin Video Unit SW
get_functional_chain The chain from Acquire Audio Stream → Decode Video Packet → Display Video on Seat TV (UUID: 07256f31-32c7-42f5-b36b-bebd43492921)
impact_analysis 7 components + 12 exchanges in the impact radius

The final response includes:

  • Likely subsystem: Cabin Video Display Unit
  • ATA chapter: 44 - Cabin Systems
  • Affected functional chain (with clickable UUID)
  • Prioritized components to inspect
  • Related requirements (REQ-IFE-047, REQ-IFE-091)
  • Recommended action: pull the Cabin Video Display Unit for row 14

The elapsed time indicator in the response shows approximately 47 seconds. That is the payoff moment.

Click-Through: Navigate to the Functional Chain

Click the UUID 07256f31-32c7-42f5-b36b-bebd43492921 in the response. The Project Explorer jumps to Display VOD Movie on Seat TV and highlights it. The associated Sirius diagram opens with that functional chain in focus.

Trust: Audit Log and Undo

Open Window → Show View → Audit Log. Every tool call is recorded: tool name, parameters, element UUIDs touched, and timestamp. Nothing is hidden.

In Capella, go to Edit → Undo. Every action the agent took is on the Eclipse undo stack. Nothing is irreversible.

Reach: Offline Mode and MCP Bridge

  • Switch the provider to Ollama (Local) in Preferences. The status bar shows "Ready — offline". The entire assistant runs locally on your laptop, suitable for ITAR or air-gapped networks.
  • In a terminal, run claude "List all physical functions from the IFE model" via the MCP bridge. All 95+ tools are available from the command line too.
