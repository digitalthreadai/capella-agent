Capella Agent Cookbook

The Cookbook contains step-by-step recipes for common MBSE tasks with Capella Agent. Each recipe is a complete walkthrough — open the sample model, type these queries, and observe the results.

All recipes use the In-Flight Entertainment (IFE) sample model bundled in the repository under demo/.

Recipes

Recipe 1: Explore a Capella Model

Learn the fundamental navigation pattern: list elements, click UUIDs to navigate, get element details, and query relationships.

Mode: General Assistant • Tools used: list_elements, get_element_details, get_functional_chain, get_interfaces

Recipe 2: Create Model Elements

Create physical components, ports, and functional exchanges using natural language. Learn how to undo multi-tool turns.

Mode: General Assistant • Tools used: create_element, create_port, create_exchange, create_functional_chain

Recipe 3: Requirements Roundtrip

Import requirements from ReqIF or Excel, link them to model elements, generate a coverage dashboard, and export back to ReqIF.

Mode: Requirements Analyst (Week 13-16) • Tools: import_reqif, link_requirements_to_elements, coverage_dashboard, export_reqif

Recipe 4: Sustainment Engineering Demo

The full 5-minute IFE demo: diagnose a field fault using the live model, trace the impact, and generate a maintenance recommendation — in under 60 seconds.

Mode: Sustainment Engineer • Tools: classify_fault_ata, lookup_fault_symptom, get_functional_chain, impact_analysis

Recipe 5: Generative Architecture

Switch to Architect mode, paste requirements, receive a full architecture proposal as a diff preview, and review missing traces.

Mode: Architect (Week 17-22) • Tools: propose_architecture, suggest_decomposition, find_missing_traces

New to Capella Agent? Start with Recipe 1 (Explore) to get comfortable with the navigation pattern before trying the write operations in Recipe 2.
