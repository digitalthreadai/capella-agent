Getting Started with Capella Agent

This guide walks you from a fresh Capella installation to your first AI-assisted model query in about 10 minutes.

Prerequisites

RequirementVersionNotes
Eclipse Capella7.0.1Available from polarsys.org
Java17 or laterMust be the JVM Capella launches with
Maven (for build)3.8+Only needed if building from source
Internet access—Required for cloud LLM providers; not needed for Ollama

Installation Methods

Method 1: reinstall.bat (Recommended for Development)

The reinstall.bat script in the repository root builds the plugin from source and side-loads it directly into your Capella installation. This is the fastest workflow when you want to keep up with the latest changes.

  1. Clone the repository: git clone https://github.com/your-org/capella-agent.git
  2. Edit reinstall.bat and set CAPELLA_HOME to your Capella installation directory (e.g., C:\Apps\capella-7.0.1).
  3. Run reinstall.bat from the repository root. It runs mvn clean verify then copies the plugin JARs into Capella's dropins folder.
  4. Restart Capella.
Tip: After the first successful build, subsequent rebuilds only take 30-60 seconds because Maven caches the compiled classes.

Method 2: Install New Software (GUI)

  1. In Capella, open Help → Install New Software...
  2. Click Add... and point to the p2 update site URL (or a local file:/// path to com.capellaagent.site/target/repository after building).
  3. Select Capella Agent - AI Multi-Agent Ecosystem from the category list.
  4. Follow the wizard. Accept the EPL 2.0 license. Restart when prompted.
Note: Plugin JARs in this beta release are not code-signed. Windows may show a SmartScreen warning. See Known Limitations #8 for the workaround.

First Launch — The Welcome Wizard

The first time you open the AI Model Chat view, a welcome wizard appears automatically. It guides you through:

  1. Selecting an LLM provider (GitHub Models is pre-selected as the free default)
  2. Entering your API key and clicking Test Connection
  3. Choosing your default agent mode (General Assistant for most users)

You can revisit these settings at any time via Window → Preferences → Capella Agent.

Opening the Chat View

Three ways to open the AI Model Chat view:

  • Keyboard shortcut: Ctrl+Shift+M
  • Menu: Window → Show View → Other... → Capella Agent → AI Model Chat
  • Toolbar: click the robot icon in the Capella toolbar (if visible)

Your First Query

  1. Open the In-Flight Entertainment (IFE) sample model included in the repository under demo/.
  2. In the chat view, make sure the IFE project is selected in the project dropdown.
  3. Type the following in the input field and press Enter: 
    List all physical functions
  4. The agent calls list_elements with layer=pa, element_type=functions and returns a sortable HTML table of all physical functions with their UUIDs, names, and descriptions.
  5. Click any UUID in the table to navigate to that element in the Project Explorer.
Next steps: Read the First Chat guide for a full walkthrough of the result table and toolbar buttons, or jump straight to the Explore a Model recipe.
