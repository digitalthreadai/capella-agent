Basic Workflow

  1. Open a Capella model — the agent can only access open Capella sessions.
  2. Select the project in the chat view toolbar if you have multiple projects open.
  3. Choose an agent mode appropriate for your current task.
  4. Type a query in natural language and press Enter.
  5. Review the response — tool calls are shown inline; model results are rendered as tables or structured text.
  6. Follow up — the agent maintains conversation context within a session.
Power user tip: Use slash commands for instant control without navigating menus. Type /help in the chat input at any time to see the full list.
